## Youtube Data API Wrapper
A simple Python module to simplify implementation of Youtube Data API client.

## Get Started
Clone this repo:
```
git clone https://github.com/pasto03/youtube-data-api-wrapper.git
```

Add .env file:
```
API_KEY=YOUR_GOOGLE_API_KEY
```

Install requirements:
```
pip install -r requirements.txt
```

Check demo in __demo__ folder, and you are good to go!

## Caution
