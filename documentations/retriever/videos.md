# VideosRetriever

## Purpose

The `VideosRetriever` retrieves detailed information about YouTube videos given their video IDs. It uses batch processing to efficiently retrieve multiple videos in a single API call.

## Input Requirements

- **Input Type**: `list[str]` — List of video IDs
- **Format**: Video IDs in standard YouTube format (e.g., `"dQw4w9WgXcQ"`)
- **Batch Size**: Automatically batches up to 50 video IDs per API call

## Output Structure

Returns a list of dictionaries, each containing:

- **Core Fields**: `kind`, `etag`, `id`
- **Snippet**: Video title, description, published date, channel information, thumbnails, tags, category
- **Content Details**: Duration, dimension, definition, caption availability, region restrictions
- **Statistics**: View count, like count, favorite count, comment count
- **Status**: Privacy status, upload status, license (if requested)
- **Topic Details**: Topic IDs (if requested)

### Key Fields

- `snippet.title` — Video title
- `snippet.description` — Video description
- `snippet.publishedAt` — Video upload date
- `snippet.channelId` — Channel ID that uploaded the video
- `snippet.channelTitle` — Channel name
- `snippet.thumbnails` — Video thumbnail images
- `snippet.tags` — Video tags
- `contentDetails.duration` — Video duration (ISO 8601 format)
- `contentDetails.definition` — Video quality (hd, sd)
- `contentDetails.caption` — Whether captions are available
- `statistics.viewCount` — Number of views
- `statistics.likeCount` — Number of likes
- `statistics.commentCount` — Number of comments

## Common Use Cases

1. **Video Analytics** — Get statistics for multiple videos
2. **Video Information** — Retrieve video details for display
3. **Batch Processing** — Process multiple videos efficiently
4. **Content Analysis** — Analyze video metadata and statistics
5. **Data Collection** — Collect video data for research or analysis

## Example Usage

### Basic Usage

```python
from yt_pipeline.retriever import VideosRetriever

video_ids = ["dQw4w9WgXcQ", "jNQXAC9IVRw"]
retriever = VideosRetriever(
    iterable=video_ids,
    developerKey=devKey
)

results = retriever.invoke()

for video in results:
    print(f"Title: {video['snippet']['title']}")
    print(f"Views: {video['statistics']['viewCount']}")
    print(f"Duration: {video['contentDetails']['duration']}")
```

## Special Considerations

1. **Batch Efficiency** — Processes up to 50 video IDs per API call automatically
2. **Quota Cost** — 1 unit per API call (regardless of number of videos, up to 50)
3. **Private Videos** — Returns data only if video is public or accessible with your API key
4. **Deleted Videos** — Returns error for deleted or unavailable videos
5. **Duration Format** — Duration is in ISO 8601 format (e.g., "PT4M13S" = 4 minutes 13 seconds)
