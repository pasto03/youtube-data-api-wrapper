# ChannelsRetriever

## Purpose

The `ChannelsRetriever` retrieves detailed information about YouTube channels given their channel IDs. It uses batch processing to efficiently retrieve multiple channels in a single API call.

## Input Requirements

- **Input Type**: `list[str]` — List of channel IDs
- **Format**: Channel IDs in standard YouTube format (e.g., `"UC_x5XG1OV2P6uZZ5FSM9Ttw"`)
- **Batch Size**: Automatically batches up to 50 channel IDs per API call

## Output Structure

Returns a list of dictionaries, each containing:

- **Core Fields**: `kind`, `etag`, `id`
- **Snippet**: Channel title, description, custom URL, published date, thumbnails, country, default language
- **Statistics**: View count, subscriber count, video count, hidden subscriber status
- **Branding Settings**: Channel branding information (if requested)
- **Content Details**: Related playlists, country (if requested)

### Key Fields

- `snippet.title` — Channel name
- `snippet.description` — Channel description
- `snippet.customUrl` — Custom channel URL (if set)
- `snippet.publishedAt` — Channel creation date
- `snippet.thumbnails` — Channel thumbnail images (default, medium, high)
- `statistics.viewCount` — Total channel views
- `statistics.subscriberCount` — Number of subscribers
- `statistics.videoCount` — Number of videos
- `statistics.hiddenSubscriberCount` — Whether subscriber count is hidden

## Common Use Cases

1. **Channel Analytics** — Get statistics for multiple channels
2. **Channel Information** — Retrieve channel details for display
3. **Batch Processing** — Process multiple channels efficiently
4. **Data Collection** — Collect channel data for analysis

## Example Usage

### Basic Usage

```python
from yt_pipeline.retriever import ChannelsRetriever

channel_ids = ["UC_x5XG1OV2P6uZZ5FSM9Ttw", "UCuAXFkgsw1L7xaCfnd5JJOw"]
retriever = ChannelsRetriever(
    iterable=channel_ids,
    developerKey=devKey
)

results = retriever.invoke()

for channel in results:
    print(f"Channel: {channel['snippet']['title']}")
    print(f"Subscribers: {channel['statistics']['subscriberCount']}")
    print(f"Videos: {channel['statistics']['videoCount']}")
```

2. **Quota Cost** — 1 unit per API call (regardless of number of channels, up to 50)
