# PlaylistsRetriever

## Purpose

The `PlaylistsRetriever` retrieves playlists from YouTube channels given channel IDs. It supports pagination to retrieve all playlists from a channel.

## Input Requirements

- **Input Type**: `list[str]` — List of channel IDs
- **Format**: Channel IDs in standard YouTube format (e.g., `"UC_x5XG1OV2P6uZZ5FSM9Ttw"`)
- **One-to-Many**: One channel ID can return multiple playlists

## Output Structure

Returns a list of dictionaries, each containing:

- **Core Fields**: `kind`, `etag`, `id`
- **Snippet**: Playlist title, description, published date, thumbnails, channel information
- **Content Details**: Item count, video count
- **Status**: Privacy status (if requested)

### Key Fields

- `id` — Playlist ID
- `snippet.title` — Playlist name
- `snippet.description` — Playlist description
- `snippet.publishedAt` — Playlist creation date
- `snippet.thumbnails` — Playlist thumbnail images
- `snippet.channelId` — Channel ID that owns the playlist
- `snippet.channelTitle` — Channel name
- `contentDetails.itemCount` — Number of items in playlist

## Common Use Cases

1. **Channel Playlists** — Get all playlists from a channel
2. **Playlist Discovery** — Find playlists by channel
3. **Content Organization** — Analyze how channels organize content
4. **Batch Processing** — Process playlists from multiple channels

## Example Usage

### Basic Usage

```python
from yt_pipeline.retriever import PlaylistsRetriever, PipeSettings

retriever = PlaylistsRetriever(
    iterable=["UC_x5XG1OV2P6uZZ5FSM9Ttw"],
    developerKey=devKey,
    settings=PipeSettings(max_page=1)  # Get first page only
)

results = retriever.invoke()

for playlist in results:
    print(f"Playlist: {playlist['snippet']['title']}")
    print(f"Items: {playlist['contentDetails']['itemCount']}")
```

### Get All Playlists

```python
retriever = PlaylistsRetriever(
    iterable=["UC_x5XG1OV2P6uZZ5FSM9Ttw"],
    developerKey=devKey,
    settings=PipeSettings(retrieval="all", max_page=50)  # Get all playlists
)

results = retriever.invoke(multithread=True)
```

### Multiple Channels

```python
channel_ids = ["UC_x5XG1OV2P6uZZ5FSM9Ttw", "UCuAXFkgsw1L7xaCfnd5JJOw"]
retriever = PlaylistsRetriever(
    iterable=channel_ids,
    developerKey=devKey,
    settings=PipeSettings(max_page=5),
    max_workers=8
)

results = retriever.invoke(multithread=True)
```

## Special Considerations

1. **Pagination** — Channels can have many playlists; use `max_page` to limit
2. **Quota Cost** — 1 unit per API call per channel
3. **Private Playlists** — Only returns public playlists or those accessible with your API key
4. **Empty Channels** — Returns empty list if channel has no playlists
