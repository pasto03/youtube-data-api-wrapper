# VideoCategoriesRetriever

## Purpose

The `VideoCategoriesRetriever` retrieves video category information for a specific region. Categories are used to classify videos on YouTube.

## Input Requirements

- **Input Type**: `list[str]` — List of region codes (ISO 3166-1 alpha-2 format, e.g., `"US"`, `"GB"`)
- **Format**: Two-letter country codes
- **One-to-Many**: One region code returns multiple categories

## Output Structure

Returns a list of dictionaries, each containing:

- **Core Fields**: `kind`, `etag`, `id`
- **Snippet**: Category title, assignable status, channel information

### Key Fields

- `id` — Category ID
- `snippet.title` — Category name (e.g., "Music", "Gaming", "Education")
- `snippet.assignable` — Whether videos can be assigned to this category
- `snippet.channelId` — Channel ID (if category is channel-specific)

## Common Use Cases

1. **Category Lookup** — Get category names from IDs
2. **Regional Categories** — Get categories available in specific regions
3. **Content Classification** — Understand video categorization
4. **Data Analysis** — Analyze category distribution

## Example Usage

### Basic Usage

```python
from yt_pipeline.retriever import VideoCategoriesRetriever, PipeSettings

retriever = VideoCategoriesRetriever(
    iterable=["US"],  # United States region
    developerKey=devKey,
    settings=PipeSettings()
)

results = retriever.invoke()

for category in results:
    print(f"ID: {category['id']}")
    print(f"Title: {category['snippet']['title']}")
    print(f"Assignable: {category['snippet']['assignable']}")
```

### Multiple Regions

```python
regions = ["US", "GB", "JP"]
retriever = VideoCategoriesRetriever(
    iterable=regions,
    developerKey=devKey,
    settings=PipeSettings()
)

results = retriever.invoke(multithread=True)
```

## Special Considerations

1. **Regional Differences** — Categories vary by region
2. **Quota Cost** — 1 unit per API call per region
3. **Assignable Status** — Some categories may not be assignable to new videos
4. **Category IDs** — Use category IDs when filtering searches or setting video categories
