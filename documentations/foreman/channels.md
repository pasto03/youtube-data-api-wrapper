# ChannelsForeman

## Purpose

The `ChannelsForeman` retrieves and structures channel data from YouTube. It orchestrates the complete workflow: retrieves channel details, parses them into structured objects, and optionally flattens them for export.

## Input Requirements

- **Input Type**: `list[str]` — List of channel IDs
- **Format**: Channel IDs in standard YouTube format (e.g., `"UC_x5XG1OV2P6uZZ5FSM9Ttw"`)

## Output Structure

### Container Output (`as_box=True`)

Returns `ChannelsContainer` with `ChannelItem` objects:

- `id` — Channel ID
- `snippet.title` — Channel name
- `snippet.description` — Channel description
- `snippet.customUrl` — Custom channel URL
- `snippet.publishedAt` — Channel creation date
- `snippet.thumbnails` — List of thumbnail objects
- `statistics.viewCount` — Total views
- `statistics.subscriberCount` — Subscriber count
- `statistics.videoCount` — Video count

### Shipper Output (`as_box=False`, default)

Returns `ChannelShipper` with:

- `main_records` — List of flattened channel dictionaries
- `thumbnails` — Dictionary mapping channel IDs to thumbnail lists

## Common Use Cases

1. **Channel Analytics** — Get structured channel statistics
2. **Channel Information Display** — Retrieve channel details for UI
3. **Batch Processing** — Process multiple channels efficiently
4. **Data Export** — Export channel data to JSON/CSV

## Example Usage

### Structured Access

```python
from yt_pipeline.foreman import ChannelsForeman
from yt_pipeline.retriever import RetrieverSettings

foreman = ChannelsForeman()
container = foreman.invoke(
    iterable=["UC_x5XG1OV2P6uZZ5FSM9Ttw"],
    developerKey=devKey,
    as_box=True
)

for channel in container.items:
    print(channel.snippet.title)
    print(channel.statistics.subscriberCount)
```

### Flattened Export

```python
foreman = ChannelsForeman()
shipper = foreman.invoke(
    iterable=["UC_x5XG1OV2P6uZZ5FSM9Ttw"],
    developerKey=devKey
)

import json
with open("channels.json", "w") as f:
    json.dump(shipper.main_records, f, indent=2)
```
