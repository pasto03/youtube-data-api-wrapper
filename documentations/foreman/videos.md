# VideosForeman

## Purpose

The `VideosForeman` retrieves and structures video data from YouTube. It orchestrates the complete workflow: retrieves video details, parses them into structured objects, and optionally flattens them for export.

## Input Requirements

- **Input Type**: `list[str]` — List of video IDs
- **Format**: Video IDs in standard YouTube format (e.g., `"dQw4w9WgXcQ"`)

## Output Structure

### Container Output (`as_box=True`)

Returns `VideosContainer` with `VideoItem` objects containing:

- `id` — Video ID
- `snippet.title` — Video title
- `snippet.description` — Video description
- `snippet.publishedAt` — Upload date
- `snippet.channelId` — Channel ID
- `snippet.channelTitle` — Channel name
- `snippet.thumbnails` — Video thumbnails
- `contentDetails.duration` — Video duration (ISO 8601)
- `contentDetails.definition` — Video quality
- `statistics.viewCount` — View count
- `statistics.likeCount` — Like count
- `statistics.commentCount` — Comment count

### Shipper Output (`as_box=False`, default)

Returns `VideosShipper` with:

- `main_records` — List of flattened video dictionaries
- `thumbnails` — Dictionary mapping video IDs to thumbnail lists

## Common Use Cases

1. **Video Analytics** — Get structured video statistics
2. **Video Information Display** — Retrieve video details for UI
3. **Batch Processing** — Process multiple videos efficiently
4. **Content Analysis** — Analyze video metadata

## Example Usage

### Structured Access

```python
from yt_pipeline.foreman import VideosForeman
from yt_pipeline.retriever import RetrieverSettings

foreman = VideosForeman()

    as_box=True
```
